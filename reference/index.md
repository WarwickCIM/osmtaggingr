# Package index

## Datasets

Datasets provided by this package

- [`proposals`](https://warwickcim.github.io/osmtaggingr/reference/proposals.md)
  : OSM Tagging Proposals Data
- [`proposals_votes`](https://warwickcim.github.io/osmtaggingr/reference/proposals_votes.md)

## Data retrieval

Functions used to retrieve data

- [`get_proposals_info()`](https://warwickcim.github.io/osmtaggingr/reference/get_proposals_info.md)
  : Scrape stats from tagging proposals
- [`get_proposals_votes()`](https://warwickcim.github.io/osmtaggingr/reference/get_proposals_votes.md)
  : Scrape votes for tagging proposals
- [`get_tagging_proposals()`](https://warwickcim.github.io/osmtaggingr/reference/get_tagging_proposals.md)
  : Retrieve a list of tagging proposals
- [`get_voting_summary()`](https://warwickcim.github.io/osmtaggingr/reference/get_voting_summary.md)
  : Scrape voting summary from tagging proposals
