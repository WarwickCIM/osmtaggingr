# OSM Tagging Proposals

Metadata tagging proposals in OSM Wiki (). This dataset has been created
by the script data-raw/tagging_proposals.R

## Usage

``` r
proposals
```

## Format

A data frame with 5 rows and 34 columns:

- id:

  A unique identifier for every boundary

- service:

  URL pointing to the API service

- url_download:

  URL querying the API service to return all features as a geojson file
